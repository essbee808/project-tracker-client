# Welcome to Project Tracker! (Flatiron School Phase 5 Porfolio Project)

Project Tracker is a fullstack web application designed for users to log and manage ideas. 

Technologies used: 
- React/Redux to build components
- React Bootstrap for styling
- Rails API [Backend Repo Here](https://github.com/essbee808/project-tracker-api)

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3001](http://localhost:3001) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
